# 🚀 快速开始指南

## 3 步生成你的第一首 AI 歌曲！

### 第 1 步：打开网站
直接双击 `index.html` 文件，或在浏览器中打开。

### 第 2 步：填写创作参数

**示例 1：欢快的夏日歌曲**
```
主题：夏日的海风
情感：😊 欢快喜悦
风格：流行 Pop
节奏：中板 (80-120 BPM)
语言：中文
```

**示例 2：忧伤的抒情歌**
```
主题：深夜的思念
情感：😢 忧伤抒情
风格：抒情 Ballad
节奏：慢板 (60-80 BPM)
语言：中文
```

**示例 3：浪漫的婚礼歌曲**
```
主题：永恒的爱
情感：💕 浪漫温馨
风格：抒情 Ballad
节奏：慢板 (60-80 BPM)
语言：中文
```

### 第 3 步：点击"开始 AI 创作"

等待 1-2 秒，你将看到：

1. **🎵 音频播放器**（页面顶部）
   - 点击 ▶️ 播放按钮聆听歌曲
   - 点击 💾 下载按钮保存 WAV 文件
   - 点击 🔄 重新生成获得不同版本

2. **📝 完整歌词**
   - 包含主歌、副歌、桥段等结构
   - 点击 📋 复制歌词
   - 点击 🔄 重新生成歌词

3. **🎼 编曲方案**
   - 和弦进行
   - 鼓点节奏
   - 乐器配置
   - 曲式结构
   - 制作要点

## 💡 小贴士

### 获得更好的结果

1. **主题要具体**
   - ✅ 好："夏日的海风"、"毕业那天"
   - ❌ 差："爱情"、"生活"

2. **尝试不同组合**
   - 同一主题 + 不同情感 = 完全不同的歌曲
   - 例如："大海" + 欢快 vs "大海" + 忧伤

3. **多次生成**
   - 每次生成的歌词和音频都不同
   - 点击"重新生成"获得更多版本
   - 选择最喜欢的一个

4. **查看编曲建议**
   - 学习专业的和弦进行
   - 了解适合的乐器配置
   - 参考制作技巧

### 音频使用说明

**当前版本**：
- 时长：30 秒
- 格式：WAV（44.1kHz, 16bit）
- 声道：立体声
- 大小：约 5 MB

**如何播放**：
- 在网页中直接播放
- 下载后用任何音乐播放器打开
- 导入到音频编辑软件（Audacity、Logic Pro 等）

**如何提升音质**：
1. 配置 Hugging Face MusicGen API（免费）
   - 详见 `AUDIO_GUIDE.md`
2. 使用 Suno AI 等专业服务（付费）
3. 将歌词和编曲交给专业音乐人制作

## 🔧 故障排除

### 问题 1：点击按钮没反应
**解决**：确保使用现代浏览器（Chrome、Firefox、Edge、Safari）

### 问题 2：音频无法播放
**解决**：
1. 检查浏览器是否允许自动播放
2. 尝试点击页面任意位置后再播放
3. 刷新页面重试

### 问题 3：音频听起来很简单
**说明**：这是浏览器端实时合成的基础版本
**改进**：配置 Hugging Face MusicGen API 获得专业音质（见 AUDIO_GUIDE.md）

### 问题 4：作品历史丢失
**原因**：清除了浏览器缓存
**预防**：重要的作品记得下载保存
