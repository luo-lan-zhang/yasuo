## 🔑 获取API Key

### OpenAI（推荐）
1. 访问 https://platform.openai.com/api-keys
2. 注册账号（新用户$5免费额度）
3. 创建API Key
4. 复制粘贴到配置面板

### Hugging Face（免费）
1. 访问 https://huggingface.co/settings/tokens
2. 注册账号（完全免费）
3. 创建Token（Read权限）
4. 复制粘贴到配置面板

### Claude
1. 访问 https://console.anthropic.com/
2. 注册账号（新用户$5免费额度）
3. 创建API Key
4. 复制粘贴到配置面板

## ❓ 常见问题

**Q: 必须配置API吗？**  
A: 不是。本地模板无需配置即可使用，API是可选的高级功能。

**Q: API失败怎么办？**  
A: 系统会自动降级到本地模板，确保始终可用。

**Q: 中国用户能用吗？**

**Q: 如何控制成本？**  
A: 设置消费限额，或主要使用免费的Hugging Face。

**Q: 生成的内容有版权吗？**  
A: 归您所有，可自由使用和商用。
