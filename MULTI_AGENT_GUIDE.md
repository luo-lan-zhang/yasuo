# 🤖 AI Music Agents - 多智能体音乐创作平台

## 📋 系统架构

本平台采用**多智能体分工协作架构**，包含4个独立智能体模块，各司其职、流程闭环。

### 四大智能体

#### 1️⃣ **作词智能体（Lyrics Agent）** ✍️
- **职责**：专业歌词创作
- **输入**：主题、情绪、风格、关键词、语言
- **输出**：标准完整歌词（主歌→副歌→桥段→尾奏）
- **特点**：押韵工整、逻辑连贯、有画面感

#### 2️⃣ **编曲智能体（Arrangement Agent）** 🎼
- **职责**：专业编曲方案设计
- **输入**：BPM、调式、歌词参考、情绪、风格
- **输出**：完整编曲方案
  - 曲风定位
  - 调式/调性
  - BPM速度
  - 和弦进行（分段落）
  - 段落乐器编排（5类乐器）
  - 节奏型、律动特点、氛围描述

#### 3️⃣ **评估智能体（Evaluation Agent）** 📊
- **职责**：作品质量评估
- **输入**：歌词、编曲方案
- **输出**：专业评估报告
  - 歌词评分（5维度）
  - 编曲评分（5维度）
  - 综合评分 + 评语
  - 优化建议

#### 4️⃣ **审核总控智能体（Audit Agent）** 🛡️
- **职责**：风控审核与最终决策
- **输入**：歌词、编曲、评估报告
- **输出**：审核结论
  - 内容安全检查
  - 逻辑一致性检查
  - 质量标准检查
  - 原创性检查
  - 最终结论：通过/修改建议

## 🔄 工作流程

```
用户输入需求
    ↓
作词智能体生成歌词
    ↓
编曲智能体生成编曲
    ↓
评估智能体打分评价
    ↓
审核智能体终审
    ↓
存入历史作品库
```

## 🎯 使用指南

### 方式一：独立使用各智能体

#### 使用作词智能体
1. 点击侧边栏"作词智能体"
2. 填写创作参数：
   - 主题（必填）
   - 情绪选择
   - 风格选择
   - 关键词（可选）
   - 语言选择
3. 点击"🚀 开始作词"
4. 查看生成的歌词
5. 可复制或保存

#### 使用编曲智能体
1. 点击侧边栏"编曲智能体"
2. 设置编曲参数：
   - BPM速度
   - 调式选择
   - 参考歌词（可选）
3. 点击"🎹 开始编曲"
4. 查看专业编曲方案
5. 可复制或保存

#### 使用评估智能体
1. 确保已生成歌词和编曲
2. 点击侧边栏"评估智能体"
3. 点击"🔍 开始评估"
4. 查看详细评估报告
5. 查看评分和优化建议

#### 使用审核智能体
1. 确保已完成前三步
2. 点击侧边栏"审核智能体"
3. 点击"🛡️ 执行审核"
4. 查看审核报告
5. 根据结论决定是否通过

### 方式二：一体化创作工作台（推荐）

1. 点击侧边栏"创作工作台"
2. 填写完整的创作需求：
   - 歌曲标题（可选）
   - 主题（必填）
   - 情绪、风格
   - BPM、调式
   - 关键词
3. 点击"🚀 启动多智能体创作流程"
4. 系统自动按顺序执行：
   - 作词 → 编曲 → 评估 → 审核
5. 在总控中心查看实时进度
6. 完成后自动跳转到成果展示
7. 审核通过的作品自动保存到历史库

## 💡 使用技巧

### 获得更好的歌词
1. **主题要具体**
   - ✅ "夏日海滩的午后"
   - ❌ "夏天"

2. **关键词要有画面感**
   - ✅ "阳光、海浪、冰淇淋"
   - ❌ "快乐、美好"

3. **情绪与风格匹配**
   - 欢快 + 流行 = 阳光 upbeat
   - 忧伤 + 抒情 = 深情 ballad

### 获得更好的编曲
1. **BPM选择**
   - 慢板抒情：60-80
   - 中板流行：80-120
   - 快板舞曲：120-160

2. **调式选择**
   - 明亮欢快：大调（C、G、D）
   - 忧伤深沉：小调（Am、Em、Dm）

3. **提供歌词参考**
   - 粘贴已生成的歌词
   - 帮助编曲智能体理解情绪

### 理解评估报告
- **9-10分**：优秀作品，可直接制作
- **8-9分**：良好作品，细节可优化
- **7-8分**：合格作品，建议改进
- **<7分**：需要大幅修改

### 应对审核不通过
1. 仔细阅读审核报告
2. 查看具体问题所在
3. 参考优化建议修改
4. 重新执行审核流程

## ❓ 常见问题

**Q: 为什么需要4个智能体？**
A: 分工协作可以提高专业性和可控性。每个智能体专注一个领域，提供更专业的结果。同时便于单独调试和优化。

**Q: 智能体是真实的AI吗？**
A: 当前版本使用规则引擎和模板系统模拟AI行为。未来计划集成真实的AI API（如GPT-4、MusicGen等）。

**Q: 数据会上传到服务器吗？**
A: 不会。所有数据处理都在本地浏览器完成，使用LocalStorage存储，完全保护隐私。

**Q: 可以商用生成的作品吗？**
A: 可以。生成的内容归您所有，可自由使用。但建议进行人工润色和优化。

**Q: 如何获得更专业的结果？**
A: 
1. 提供详细具体的创作参数
2. 使用评估报告的建议进行优化
3. 多次生成，选择最佳版本
4. 结合人工专业知识进行调整

**Q: 支持哪些音乐风格？**
A: 目前支持6种主流风格：流行、摇滚、抒情、R&B、民谣、电子。未来会添加更多风格。

**Q: 历史作品会丢失吗？**
A: 只要不清除浏览器缓存，作品会永久保存。建议定期导出重要作品备份。

**Q: 可以在手机上使用吗？**
A: 可以。平台采用响应式设计，完美适配移动端。但大屏幕有更好的编辑体验。

**Q: 如何导出作品？**
A: 
1. 在成果展示页点击"复制全部"
2. 或点击"下载"按钮保存为TXT文件
3. 也可在各个智能体页面单独复制

**Q: 审核不通过怎么办？**
A: 
1. 查看审核报告中的具体问题
2. 根据建议修改歌词或编曲
3. 重新执行审核流程
4. 或在独立智能体中手动调整
