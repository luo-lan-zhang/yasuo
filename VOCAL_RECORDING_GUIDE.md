# 🎤 人声录制功能使用指南

## ✨ 功能说明

平台现已支持**录制你的人声**并**与伴奏合并**，让你创作出完整的歌曲作品！

### 核心功能

1. **🎙️ 人声录制** - 使用麦克风录制你的歌声
2. **📊 实时可视化** - 录音时显示音频波形
3. **▶️ 即时回放** - 录完立即播放检查
4. **💾 下载人声** - 单独下载你的人声录音
5. **🎼 混音合并** - 将人声与伴奏混合成完整歌曲
6. **🔄 重新录制** - 不满意可以多次重录

## 🚀 快速开始

### 第1步：完成创作流程
1. 使用多智能体系统创作歌词和编曲
2. 等待生成最终编曲录音
3. 进入"成果展示"页面

### 第2步：准备录音环境
- ✅ 佩戴耳机（避免回音）
- ✅ 安静的环境
- ✅ 良好的麦克风
- ✅ 熟悉歌词内容

### 第3步：开始录制
1. 找到"🎤 人声录制工作室"模块
2. 点击红色"● 开始录制"按钮
3. 允许浏览器访问麦克风
4. 对着麦克风演唱
5. 点击橙色"■ 停止"按钮结束

### 第4步：检查和调整
1. 点击绿色"▶ 播放"按钮试听
2. 满意则继续下一步
3. 不满意点击"🔄 重新录制"

### 第5步：合并成完整歌曲
1. 点击"🎼 合并到伴奏"按钮
2. 等待系统处理（约3-5秒）
3. 播放混音后的完整歌曲
4. 点击"💾 下载完整歌曲"保存

## 💡 录音技巧

### 设备准备
**推荐配置：**
- 耳机：任何有线耳机（避免蓝牙延迟）
- 麦克风：手机耳机麦 > 笔记本内置麦 > USB麦克风
- 环境：安静房间，关闭空调/风扇

**不推荐：**
- ❌ 蓝牙耳机（有延迟）
- ❌ 嘈杂环境
- ❌ 空旷房间（回音大）

### 录音姿势
1. **距离麦克风**：10-15厘米
2. **角度**：稍微偏离正前方（减少喷麦）
3. **姿势**：坐直或站直，保持呼吸顺畅
4. **音量**：正常说话音量的1.5倍

### 演唱技巧
1. **热身**：录音前唱几句热身
2. **呼吸**：深吸气，控制气息
3. **节奏**：跟着心中的节拍唱
4. **情感**：投入感情，不要机械
5. **多次尝试**：录3-5遍选最好的

### 常见问题解决

**问题1：声音太小**
- 靠近麦克风
- 提高演唱音量
- 检查系统音量设置

**问题2：有回音**
- 必须佩戴耳机
- 在较小房间录音
- 用被子/窗帘吸收回音

**问题3：有噪音**
- 关闭空调/风扇
- 远离电脑主机
- 选择安静时段

**问题4：不同步**
- 先听几遍伴奏熟悉节奏
- 心中默数拍子
- 可以分段录制

## 🎼 混音说明

### 自动混音参数
系统会自动调整：
- **人声音量**：80%（突出人声）
- **伴奏音量**：60%（衬托人声）
- **平衡**：立体声混合
- **格式**：WAV 44.1kHz 16bit

### 混音流程
```
1. 读取人声录音
   ↓
2. 读取伴奏音频
   ↓
3. 对齐时间轴
   ↓
4. 调整音量比例
   ↓
5. 混合双声道
   ↓
6. 生成完整歌曲
   ↓
7. 提供播放和下载
```

### 音质说明
- **采样率**：44.1kHz（CD音质）
- **位深度**：16bit
- **声道**：立体声
- **格式**：WAV（无损）
- **文件大小**：约10MB/分钟

## 🔧 技术细节

### 浏览器API
- **MediaRecorder API** - 录音核心
- **Web Audio API** - 音频处理和混音
- **Canvas API** - 音频可视化
- **getUserMedia** - 麦克风访问

### 录音参数
```javascript
{
    audio: {
        echoCancellation: true,    // 回声消除
        noiseSuppression: true,    // 降噪
        sampleRate: 44100          // CD音质
    }
}
```

### 混音算法
```javascript
// 人声 80% + 伴奏 60%
mixed[i] = vocal[i] * 0.8 + backing[i] * 0.6;

// 限制在 -1 到 1 之间
mixed[i] = Math.max(-1, Math.min(1, mixed[i]));
```

### 兼容性
- ✅ Chrome 60+
- ✅ Firefox 55+
- ✅ Safari 11+（需要用户授权）
- ✅ Edge 79+
- ⚠️ 需要HTTPS或localhost
- ⚠️ 需要麦克风权限

## ❓ 常见问题

### Q: 为什么无法录音？
A: 
1. 检查浏览器是否允许麦克风权限
2. 确认设备有麦克风
3. 尝试刷新页面重新授权
4. 检查系统隐私设置

### Q: 录音质量很差怎么办？
A:
1. 使用外接麦克风或耳机麦
2. 在安静环境录音
3. 靠近麦克风（10-15cm）
4. 避免对着麦克风吹气

### Q: 可以导入外部音频吗？
A: 当前版本只支持实时录制。未来版本会添加上传功能。

### Q: 录音时长有限制吗？
A: 建议控制在3-5分钟内。理论上无限制，但文件会很大。

### Q: 可以同时听到伴奏吗？
A: 当前版本需要先听熟伴奏再录。未来会添加边听边录功能。

### Q: 混音后人声太小/太大？
A: 当前是固定比例（人声80%，伴奏60%）。未来会添加音量调节滑块。

### Q: 可以在手机上使用吗？
A: 可以！移动端浏览器也支持录音功能。建议横屏使用。

### Q: 录错可以修改吗？
A: 可以点击"重新录制"完全重录。暂不支持剪辑编辑。

### Q: 导出的文件格式是什么？
A: WAV格式，无损音质，所有播放器都支持。

### Q: 录音数据会上传吗？
A: 不会！所有处理都在本地浏览器完成，完全隐私安全。

### Q: 如何获得更好的录音效果？
A:
1. 使用外置USB麦克风
2. 在铺有地毯/窗帘的房间
3. 录音时关门窗
4. 佩戴封闭式耳机
5. 录音前热身嗓子

### Q: 可以和朋友的录音合并吗？
A: 当前版本不支持。可以先分别录制，然后用Audacity等软件合并。

### Q: 为什么可视化不动？
A: 
1. 确认已点击"开始录制"
2. 检查浏览器控制台是否有错误
3. 尝试刷新页面

### Q: 混音失败怎么办？
A:
1. 确认已完成伴奏生成
2. 确认已成功录制人声
3. 刷新页面重试
4. 检查浏览器是否支持Web Audio API

## 🎓 进阶技巧

### 分段录制法
如果整首歌唱不好：
1. 先录主歌部分
2. 停止，休息
3. 再录副歌部分
4. 用音频软件拼接

### 多层录音法
专业做法：
1. 第1遍：主旋律
2. 第2遍：和声（小声）
3. 第3遍：高音点缀
4. 用软件混合多层

### 情感表达
1. **理解歌词**：读懂每句的意思
2. **想象场景**：脑海中构建画面
3. **代入角色**：成为歌中主角
4. **自然流露**：不要刻意表演

### 节奏把握
1. **数拍子**：1-2-3-4循环
2. **打拍子**：脚轻轻点地
3. **听鼓点**：跟随节奏乐器
4. **提前半拍**：稍微超前更有力

## 🎯 最佳实践流程

### 标准流程
```
1. 创作歌词和编曲
   ↓
2. 反复听伴奏（至少10遍）
   ↓
3. 跟着伴奏练习（不录）
   ↓
4. 正式录制第1遍
   ↓
5. 回放检查
   ↓
6. 不满意 → 重新录制
   ↓
7. 满意 → 合并到伴奏
   ↓
8. 听完整歌曲
   ↓
9. 下载保存
```

### 时间分配
- **熟悉伴奏**：15-30分钟
- **练习演唱**：10-20分钟
- **正式录制**：5-10分钟（多遍）
- **选择最佳**：5分钟
- **混音导出**：1分钟
