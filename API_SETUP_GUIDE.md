## 🔧 配置步骤

### 方式一：OpenAI GPT-4（推荐）

#### 1. 获取API Key
1. 访问 [OpenAI官网](https://platform.openai.com/api-keys)
2. 注册/登录账号
3. 点击 "Create new secret key"
4. 复制API Key（格式：`sk-...`）

#### 2. 充值（可选）
- 新用户有$5免费额度
- 后续需要绑定信用卡
- 按使用量付费（约$0.03/次歌词生成）

#### 3. 在平台配置
1. 点击右上角 ⚙️ 按钮
2. 选择 "OpenAI GPT-4"
3. 粘贴API Key
4. 选择模型（GPT-4推荐）
5. 点击"保存配置"
6. 点击"测试连接"验证

#### 优势
- ✅ 质量最高，歌词专业水准
- ✅ 理解能力强，符合需求
- ✅ 押韵工整，结构完整
- ✅ 创意丰富，不落俗套

---

### 方式二：Anthropic Claude 3

#### 1. 获取API Key
1. 访问 [Anthropic官网](https://console.anthropic.com/)
2. 注册/登录账号
3. 进入API Keys页面
4. 创建新的API Key
5. 复制Key（格式：`sk-ant-...`）

#### 2. 充值
- 新用户有$5免费额度
- 后续按使用量付费（约$0.05/次）

#### 3. 在平台配置
1. 点击右上角 ⚙️ 按钮
2. 选择 "Claude 3 Opus"
3. 粘贴API Key
4. 选择模型版本
5. 保存并测试

#### 优势
- ✅ 创意性极强
- ✅ 文笔优美，富有诗意
- ✅ 情感表达细腻
- ✅ 长文本连贯性好

---

### 方式三：Hugging Face（免费）

#### 1. 获取Access Token
1. 访问 [Hugging Face](https://huggingface.co/settings/tokens)
2. 注册/登录账号（完全免费）
3. 点击 "New token"
4. 选择 "Read" 权限
5. 复制Token（格式：`hf_...`）

#### 2. 在平台配置
1. 点击右上角 ⚙️ 按钮
2. 选择 "Hugging Face"
3. 粘贴Token
4. 选择模型
5. 保存并测试

#### 优势
- ✅ 完全免费（每月20k tokens）
- ✅ 无需信用卡
- ✅ 开源社区支持
- ⚠️ 质量中等，适合学习

---

### 方式四：本地模板引擎（默认）

#### 无需配置
- 开箱即用
- 完全免费

## 🔒 安全说明

### API Key存储
- ✅ 仅存储在浏览器LocalStorage
- ✅ 不会上传到任何服务器
- ✅ 其他网站无法访问
- ⚠️ 清除浏览器数据会丢失

### 隐私保护
- 所有数据处理在本地完成
- API调用直接从你的浏览器到AI服务商
- 我们不收集、不存储任何API Key
- 建议定期更换API Key

### 使用建议
1. 不要分享API Key给他人
2. 设置使用限额防止超额
3. 定期检查API使用记录
4. 如怀疑泄露，立即撤销Key

## ❓ 常见问题

### Q: 必须配置API吗？
A: 不是。平台内置了本地模板引擎，无需配置即可使用。API是可选的高级功能。

### Q: 哪个API最好？
A: 
- **质量优先**：OpenAI GPT-4 或 Claude 3 Opus
- **性价比**：Hugging Face（免费）
- **速度优先**：本地模板

### Q: API失败怎么办？
A: 系统会自动降级到本地模板引擎，确保始终可用。

### Q: 可以同时配置多个API吗？
A: 可以配置多个，但同一时间只能启用一个。可随时切换。

### Q: 如何查看API使用情况？
A: 
- OpenAI: https://platform.openai.com/usage
- Anthropic: https://console.anthropic.com/usage
- Hugging Face: https://huggingface.co/settings/billing

### Q: 学生有优惠吗？
A: 
- OpenAI: 无学生优惠
- Anthropic: 联系销售可能有教育折扣
- Hugging Face: 完全免费

### Q: 中国用户可以正常使用吗？
A: 
- OpenAI: 需要海外支付方式
- Anthropic: 同上
- Hugging Face: 可直接使用，推荐

### Q: API生成需要多长时间？
A: 
- OpenAI GPT-4: 5-15秒
- Claude 3: 8-20秒
- Hugging Face: 3-10秒
- 本地模板: <1秒

### Q: 生成的内容有版权吗？
A: 
- OpenAI: 归用户所有，可商用
- Anthropic: 归用户所有，可商用
- Hugging Face: 视具体模型而定
- 本地模板: 归用户所有

### Q: 如何控制成本？
A: 
1. 设置API平台的消费限额
2. 先用本地模板测试想法
3. 只对满意的作品使用API
4. 监控使用记录

### Q: 可以在手机上配置吗？
A: 可以。API配置界面完全响应式，支持移动端。

### Q: 配置后会一直有效吗？
A: 是的。配置保存在浏览器中，除非清除数据，否则永久有效。

## 📞 技术支持

### 遇到问题？

1. **检查网络连接**
   - 确保能访问API服务商
   - 某些地区可能需要特殊网络

2. **验证API Key**
   - 确认Key未过期
   - 确认有足够余额

3. **查看控制台**
   - 按F12打开开发者工具
   - 查看Console中的错误信息
   - 截图反馈问题

4. **联系支持**
   - 提供错误截图
   - 说明操作步骤
   - 附上API类型
