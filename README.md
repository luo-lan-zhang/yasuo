# AI Music Studio - 智能音乐创作助手

一个专业级的AI音乐创作网站，能够自动生成高质量歌词和专业编曲建议。

## ✨ 功能特性

### 🎵 智能歌词生成
- 支持6种情感基调：欢快、忧伤、浪漫、激情、忧郁、希望
- 支持6种音乐风格：流行、摇滚、抒情、R&B、民谣、电子
- 支持中英文双语
- 智能模板系统，每次生成都不同

### 🎼 专业编曲方案
- **和弦进行**：根据情感和风格推荐专业和弦走向
- **鼓点节奏**：提供详细的鼓组编排模式
- **乐器配置**：推荐适合的乐器组合
- **曲式结构**：标准的歌曲结构安排
- **制作要点**：专业的混音和制作技巧

### 💾 作品管理
- 自动保存创作历史
- 随时查看和重新加载之前的作品
- 本地存储，保护隐私

### 🎨 大厂级UI设计
- 深色主题，专业音乐软件风格
- 玻璃态效果（Glassmorphism）
- 流畅的动画和过渡效果
- 音频可视化加载动画
- 完全响应式设计

## 🚀 快速开始

### 方法1：直接打开（推荐）
直接双击 `index.html` 文件即可在浏览器中打开使用。

### 方法2：本地服务器
```bash
# Windows
start.bat

# Mac/Linux
python3 -m http.server 8080
```

然后在浏览器中访问：http://localhost:8080

## 📖 使用指南

1. **填写创作参数**
   - 输入歌曲主题（必填）
   - 选择情感基调
   - 选择音乐风格
   - 选择节奏速度
   - 选择语言

2. **点击"开始 AI 创作"**
   - 系统会模拟AI处理过程
   - 等待1-2秒即可看到结果

3. **查看结果**
   - 左侧显示生成的完整歌词
   - 右侧显示专业编曲方案
   - 可以复制歌词或重新生成

4. **管理作品**
   - 所有作品自动保存到"我的作品"
   - 点击作品卡片可以快速加载

## 🎯 技术特点

- **纯前端实现**：无需后端服务器，开箱即用
- **智能模板系统**：基于情感和风格的智能歌词生成
- **本地存储**：使用 localStorage 保存作品历史
- **响应式设计**：完美支持桌面和移动设备
- **零依赖**：不需要安装任何依赖包

## 🎨 设计理念

采用现代音乐制作软件的深色主题设计，结合玻璃态视觉效果，打造专业且富有科技感的用户体验。流畅的动画和精心设计的交互细节，让创作过程更加愉悦。

## 📝 示例

### 输入
- 主题：夏日的海风
- 情感：欢快喜悦
- 风格：流行
- 节奏：中板

### 输出
系统会生成一首关于夏日海风的欢快流行歌曲，包含完整的歌词结构和专业的编曲建议。

## 🔧 自定义扩展

如需添加更多歌词模板，可以编辑 `script.js` 中的 `getAdvancedTemplates()` 方法，按照现有格式添加新的模板即可。
