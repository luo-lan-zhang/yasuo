# 🎵 AI Music Pro - 专业级AI歌词创作与智能编曲平台

## ✨ 产品概述

AI Music Pro 是一款专业级的音乐创作SAAS平台，集成了AI歌词生成和智能编曲功能。采用高端UI设计，提供完整的创作、编辑、导出工作流。

## 🎯 核心功能

### 1. AI 歌词生成
- **主题输入** - 设定歌曲核心主题
- **风格选择** - 8种音乐风格（流行、摇滚、抒情、R&B、民谣、电子、爵士、古典）
- **情绪设定** - 8种情感基调（欢快、忧伤、浪漫、激情、忧郁、希望、愤怒、平静）
- **关键词** - 自定义关键词融入歌词
- **长度控制** - 简短/中等/较长三种长度

### 2. 智能编曲生成
- **基础参数** - BPM、调式、拍号设置
- **和弦进行** - 根据风格和情绪自动生成专业和弦
- **完整乐器编排**:
  - 🥁 **鼓组 Drums** - 节奏型设计
  - 🎸 **贝斯 Bass** - 低音线条
  - 🎹 **和声乐器 Harmony** - 钢琴、吉他等
  - 🎺 **旋律乐器 Melody** - 主旋律配置
  - ✨ **色彩乐器 Color** - 合成器、弦乐等
- **曲式结构** - 完整的段落安排（Intro → Verse → Chorus → Bridge → Outro）

### 3. 专业编辑器
- **歌词编辑器**
  - 富文本编辑区
  - 快速添加主歌/副歌/桥段
  - 实时保存
  
- **编曲编辑器**
  - 修改BPM、调式、拍号
  - 编辑和弦进行
  - 分别编辑每个乐器部分
  - 可视化曲式结构编辑器
  - 添加/删除段落

### 4. 导出功能
- **TXT格式** - 完整的文本格式作品
- **JSON格式** - 结构化数据，便于程序处理
- **一键复制** - 快速复制到剪贴板

### 5. 历史记录
- 自动保存所有创作
- 最多保存50条历史
- 点击即可加载历史作品
- 显示创作时间和标签

### 6. 深色/浅色模式
- 一键切换主题
- 自动保存偏好
- 优雅的过渡动画

## 🚀 使用指南

### 快速开始（3步创作）

#### 第1步：填写创作参数
```
标题：夏日的风（可选）
主题：夏日海滩
风格：流行 Pop
情绪：欢快 Happy
BPM：120
调式：C 大调
关键词：阳光、海浪、自由
长度：中等
```

#### 第2步：点击"开始 AI 创作"
- 等待1-2秒
- AI自动生成歌词和编曲方案
- 在右侧预览区域查看结果

#### 第3步：编辑和完善
- 点击"编辑"按钮进入编辑器
- 修改歌词或编曲细节
- 点击"保存"按钮保存修改

### 高级功能

#### 歌词编辑技巧
1. **添加段落**
   - 点击"➕ 添加主歌"插入新的主歌部分
   - 点击"➕ 添加副歌"插入副歌
   - 点击"➕ 添加桥段"插入桥段

2. **格式化建议**
   ```
   【主歌】
   第一段歌词
   
   【副歌】
   副歌歌词
   
   【主歌2】
   第二段歌词
   ```

#### 编曲编辑技巧
1. **和弦进行格式**
   ```
   C - G - Am - F | C - G - F - G
   ```
   使用 `|` 分隔不同小节

2. **乐器描述要点**
   - **鼓组**: 描述节奏型、强调的拍子
   - **贝斯**: 描述音型、演奏技巧
   - **和声**: 说明具体乐器和演奏方式
   - **旋律**: 主奏乐器和独奏段落
   - **色彩**: 氛围性乐器和音效

3. **曲式结构编辑**
   - 点击"➕ 添加段落"增加新段落
   - 点击"✕"删除不需要的段落
   - 直接编辑段落名称

#### 导出最佳实践
1. **TXT格式** - 适合打印、分享、存档
2. **JSON格式** - 适合导入其他程序、数据分析
3. **复制功能** - 快速粘贴到聊天、文档

## 💡 创作建议

### 歌词创作
1. **主题要具体**
   - ✅ "夏日的海风"、"毕业那天"
   - ❌ "爱情"、"生活"

2. **情绪匹配**
   - 欢快的主题选"Happy"
   - 失恋的主题选"Sad"
   - 表白的主题选"Romantic"

3. **风格选择**
   - 主流歌曲 → Pop
   - 激烈表达 → Rock
   - 深情告白 → Ballad
   - 都市感 → R&B
   - 叙事性 → Folk

### 编曲配置
1. **BPM参考**
   - 慢板抒情: 60-80 BPM
   - 中板流行: 80-120 BPM
   - 快板舞曲: 120-160 BPM

2. **调式选择**
   - 明亮欢快 → 大调（C、G、D）
   - 忧伤深沉 → 小调（Am、Em、Dm）

3. **乐器搭配原则**
   - 保持频率平衡（低、中、高频都有）
   - 不要过多乐器同时演奏
   - 留出空间给人声
