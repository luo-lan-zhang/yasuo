## 🎼 音频生成技术

### Web Audio API 实时合成
- **时长**：30秒
- **格式**：WAV (44.1kHz, 16bit, 立体声)
- **大小**：约5MB

### 智能音频参数
根据情绪和风格自动调整：

| 情绪 | 基频 | 音阶 | 亮度 |
|------|------|------|------|
| 欢快 | 440Hz | 大调 | 80% |
| 忧伤 | 330Hz | 小调 | 40% |
| 浪漫 | 392Hz | 大调 | 60% |
| 激情 | 523Hz | 大调 | 90% |
| 忧郁 | 294Hz | 小调 | 30% |
| 希望 | 466Hz | 大调 | 70% |

### 多音轨合成
1. **旋律音轨**
   - 基于音阶生成随机旋律
   - 立体声panning（左右声道）
   - ADSR包络（Attack-Decay-Sustain-Release）

2. **鼓组节奏**
   - Kick Drum（底鼓）：节拍1和3
   - Snare（军鼓）：节拍2和4
   - Hi-hat（踩镲）：每八分音符

3. **环境噪音**
   - 轻微的随机噪音增加真实感
   - 模拟录音室环境

## 🎮 使用方法

### 播放音频
1. 完成创作流程
2. 自动跳转到成果展示页
3. 音频播放器自动加载
4. 点击播放按钮即可聆听

### 下载音频
1. 点击"💾 下载音频"按钮
2. 自动下载WAV文件
3. 文件名：`歌曲标题.wav`
4. 可用任何音频播放器打开

### 重新生成
1. 对当前音频不满意
2. 点击"🔄 重新生成"按钮
3. 系统生成新的音频版本
4. 自动替换并播放

## ❓ 常见问题

**Q: 为什么音频只有30秒？**
A: 为了平衡生成速度和文件大小。未来版本会支持更长音频。

**Q: 音频质量如何？**
A: 当前是Web Audio API合成的基础版本。如需专业音质，可配置AI API。

**Q: 可以商用吗？**
A: 生成的音频归您所有，可自由使用。但建议用于demo或创意参考。

**Q: 为什么有时听起来简单？**
A: 这是浏览器端实时合成的限制。真实AI模型会生成更复杂的音乐。

**Q: 如何提高音质？**
A: 
1. 配置Hugging Face MusicGen API
2. 使用Suno AI等专业服务
3. 将编曲方案交给专业音乐人制作

**Q: 下载的音频能在手机上播放吗？**
A: 可以。WAV是通用格式，所有设备都支持。

**Q: 可以修改音频吗？**
A: 当前版本不支持编辑。可下载后用Audacity等软件编辑。

**Q: 重新生成的音频会不同吗？**
A: 是的。每次生成都使用随机种子，会产生不同的旋律和节奏。
