# AI Music Pro - 专业级AI音乐创作平台

> 🎵 AI歌词生成 + 智能编曲 | 高端SAAS设计 | 纯前端实现

## ✨ 特性

- 🤖 **AI歌词生成** - 8种风格 × 8种情绪，智能生成高质量歌词
- 🎼 **智能编曲** - 完整的5大类乐器编排（鼓组、贝斯、和声、旋律、色彩）
- ✏️ **专业编辑器** - 歌词编辑 + 编曲编辑，完全可控
- 💾 **多格式导出** - TXT、JSON格式，一键复制
- 📚 **历史记录** - 自动保存，随时回溯
- 🌓 **主题切换** - 深色/浅色模式，优雅过渡
- 📱 **响应式设计** - 完美适配移动端
- 🔒 **隐私安全** - 纯本地运行，数据不上传

## 🚀 快速开始

1. 打开 `index.html`
2. 填写创作参数（主题、风格、情绪等）
3. 点击"开始 AI 创作"
4. 查看生成的歌词和编曲方案
5. 使用编辑器进行修改和完善
6. 导出为TXT或JSON文件

## 🎯 核心功能

### 1. 歌词生成
- 支持8种音乐风格
- 支持8种情感基调
- 关键词融入
- 长度控制

### 2. 编曲生成
- BPM和调式设置
- 专业和弦进行
- 5类乐器编排：
  - 🥁 鼓组
  - 🎸 贝斯
  - 🎹 和声乐器
  - 🎺 旋律乐器
  - ✨ 色彩乐器
- 曲式结构规划

### 3. 编辑功能
- 富文本歌词编辑器
- 可视化编曲编辑器
- 段落管理
- 实时保存

### 4. 导出功能
- TXT格式（可读性强）
- JSON格式（结构化）
- 一键复制
