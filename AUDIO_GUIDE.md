# 🎵 AI Music Studio - 完整歌曲生成指南

## ✨ 新功能：完整歌曲生成

现在你可以生成**包含歌词和伴奏音频的完整歌曲**了！

### 🎯 功能说明

1. **智能歌词生成** - 根据主题、情感、风格自动生成高质量歌词
2. **专业编曲方案** - 提供和弦、节奏、乐器等专业建议
3. **🆕 音频生成** - 自动生成可播放的歌曲伴奏（WAV格式）
4. **音频播放器** - 内置美观的音乐播放器，支持播放/暂停/进度控制
5. **下载功能** - 可以将生成的歌曲下载到本地

### 🚀 使用步骤

#### 1. 填写创作参数
- **歌曲主题**：例如"夏日的海风"、"深夜的思念"
- **情感基调**：欢快、忧伤、浪漫、激情、忧郁、希望
- **音乐风格**：流行、摇滚、抒情、R&B、民谣、电子
- **节奏速度**：慢板、中板、快板
- **语言**：中文或英文

#### 2. 点击"开始 AI 创作"
系统会：
- 生成完整的歌词（包含主歌、副歌、桥段等结构）
- 提供专业的编曲建议
- **自动生成 30 秒的歌曲伴奏音频**

#### 3. 欣赏你的歌曲
在页面顶部会出现一个精美的音频播放器：
- 🎵 **专辑封面动画** - 跳动的音符动画
- ▶️ **播放/暂停** - 点击播放按钮聆听歌曲
- 💾 **下载音频** - 将歌曲保存为 WAV 文件
- 🔄 **重新生成** - 不满意可以重新生成不同的版本
- 📊 **进度条** - 显示播放进度，可以点击跳转

### 🎼 音频生成技术

#### 当前实现
网站使用 **Web Audio API** 实时合成音频：
- 根据情感基调调整音色（欢快的明亮、忧伤的柔和等）
- 根据音乐风格调整节奏和旋律
- 生成 30 秒的立体声 WAV 音频
- 完全在浏览器中完成，无需服务器

#### 音频特点
- ✅ 根据情感自动调整音调
- ✅ 根据风格自动调整节奏
- ✅ 立体声输出
- ✅ 标准 WAV 格式（44.1kHz, 16bit）
- ✅ 可下载到本地使用

### 🔧 升级到真实 AI 音频（可选）

如果你想使用真正的 AI 音乐生成（类似 Suno AI），可以按照以下步骤配置：

#### 方法 1: Hugging Face MusicGen（推荐）

1. **获取 API Token**
   - 访问 https://huggingface.co/settings/tokens
   - 注册/登录账号
   - 创建一个新的 Access Token

2. **修改代码**
   在 `script.js` 中找到 `fetchAudioFromAPI` 方法，取消注释并填入你的 Token：

```javascript
async fetchAudioFromAPI(theme, emotion, style, lyrics) {
  const API_TOKEN = 'your_huggingface_token_here'; // 填入你的 Token
  const response = await fetch(
    'https://api-inference.huggingface.co/models/facebook/musicgen-small',
    {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${API_TOKEN}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        inputs: `${emotion} ${style} song about ${theme}`,
        parameters: {
          max_length: 256
        }
      })
    }
  );
  
  const blob = await response.blob();
  return URL.createObjectURL(blob);
}
```

3. **优势**
   - 真实的 AI 生成音乐
   - 更高质量的音频
   - 更符合描述的风格

#### 方法 2: Suno API（商业级）

Suno 是目前最强大的 AI 音乐生成工具，但需要付费订阅。

### 📱 播放器功能详解

#### 播放控制
- **播放/暂停按钮** - 控制音频播放
- **进度条** - 显示播放进度，点击可跳转到指定位置
- **时间显示** - 显示当前时间和总时长

- **下载按钮** - 将音频保存为 WAV 文件到本地

#### 视觉反馈
- **专辑封面** - 带有浮动音符动画的渐变色方块
- **播放状态** - 播放时按钮会有脉冲动画效果
- **进度指示** - 渐变色的进度条，悬停显示拖动手柄

### 💡 使用技巧

1. **尝试不同组合**
   - 同一主题 + 不同情感 = 完全不同的歌曲
   - 例如："大海" + 欢快 vs "大海" + 忧伤

2. **多次生成**
   - 每次生成的音频都不同
   - 点击"重新生成"获得更多版本
   - 选择最喜欢的一个下载

3. **配合编曲建议**
   - 查看右侧的编曲方案
   - 了解歌曲的和弦进行
   - 学习专业的制作技巧

4. **导出使用**
   - 下载的 WAV 文件可以在任何音频软件中打开
   - 可以导入到 DAW（数字音频工作站）进行进一步编辑
   - 可以作为 demo 分享给朋友

### ❓ 常见问题

**Q: 为什么音频只有 30 秒？**
A: 这是演示版本的限制。使用真实的 AI API（如 MusicGen）可以生成更长的音频。

**Q: 音频质量如何？**
A: 当前使用 Web Audio API 合成的音频是基础版本。如需专业级音质，建议配置 Hugging Face MusicGen API。

**Q: 可以商用吗？**
A: 当前生成的音频可以用于个人项目。商业用途建议使用授权的 AI 音乐服务。

**Q: 为什么有时音频听起来很简单？**
A: 这是浏览器端实时合成的限制。真实的 AI 模型会生成更复杂、更丰富的音乐。

**Q: 如何获得更好的音频？**
A: 
1. 配置 Hugging Face MusicGen API（免费）
2. 使用 Suno AI 等专业服务（付费）
3. 将生成的歌词和编曲建议交给专业音乐人制作
