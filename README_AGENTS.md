# 🤖 AI Music Agents - 多智能体音乐创作平台

> 4大智能体分工协作 · 流程闭环 · 专业级音乐创作

## ✨ 核心特性

- 🤖 **4个独立智能体** - 作词、编曲、评估、审核，各司其职
- 🔄 **完整工作流** - 用户需求 → 作词 → 编曲 → 评估 → 审核 → 存档
- 📊 **专业评估系统** - 10维度评分，客观专业的质量评估
- 🛡️ **风控审核机制** - 内容安全、逻辑一致性、质量标准、原创性检查
- 💾 **本地存储** - LocalStorage持久化，最多50个作品
- 🎨 **现代UI设计** - 深色主题、渐变色彩、流畅动画
- 📱 **响应式设计** - 完美适配桌面和移动端
- 🔒 **隐私安全** - 纯前端实现，数据不上传

## 🚀 快速开始

### 方式一：一体化创作（推荐）
1. 打开 `index.html`
2. 点击"创作工作台"
3. 填写创作需求
4. 点击"启动多智能体创作流程"
5. 等待自动完成所有步骤
6. 查看成果展示

### 方式二：独立使用智能体
1. 选择任意智能体模块
2. 填写对应参数
3. 点击生成按钮
4. 查看结果

## 🎯 四大智能体

### 1. 作词智能体 ✍️
- 接收：主题、情绪、风格、关键词
- 输出：标准完整歌词（主歌→副歌→桥段→尾奏）
- 特点：押韵工整、有画面感

### 2. 编曲智能体 🎼
- 接收：BPM、调式、歌词参考
- 输出：专业编曲方案
  - 和弦进行（分段落）
  - 5类乐器编排
  - 节奏型、氛围描述

### 3. 评估智能体 📊
- 接收：歌词、编曲
- 输出：评估报告
  - 10维度评分（每项1-10分）
  - 综合评语
  - 优化建议

### 4. 审核智能体 🛡️
- 接收：歌词、编曲、评估
- 输出：审核结论
  - 4项安全检查
  - 通过/修改建议
  - 最终决策
